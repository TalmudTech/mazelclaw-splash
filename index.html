Quick Start

$ git clone https://github.com/TalmudTech/mazelclaw.git
$ cd mazelclaw && pnpm install && pnpm run build
$ pnpm run openclaw onboard

What It Does

Runs on your machine

Mac, Linux, or cloud VM. Keep your context and data under your control.

Works in chat apps

Telegram, WhatsApp, Discord, Slack, Signal, and more.

Actually executes tasks

Browser automation, shell commands, file ops, tools, and background agents.

Persistent memory

Remembers preferences, context, and project history across sessions.

Skills + plugins

Install existing skills or build custom ones for your exact workflow.

Built for power users

Hackable architecture with clean CLI workflows and extensible tooling.
