신뢰 경계

1
공급망
ClawHub
  • 스킬 게시 (시맨틱 버전, SKILL.md 필수)
  • 패턴 기반 검수 플래그
  • VirusTotal Code Insight
  • GitHub 계정 생성 일자 확인
2
채널 접근 제어
Gateway
  • 기기 페어링 (30초 유예)
  • AllowFrom / AllowList 검증
  • 토큰/비밀번호/Tailscale 인증
3
세션 격리
Agent Sessions
  • 세션 키 = agent:channel:peer
  • 에이전트별 도구 정책
  • 대화 기록 로깅
4
도구 실행
Execution Sandbox
  • Docker 샌드박스 또는 호스트 (실행 승인)
  • Node 원격 실행
  • SSRF 보호 (DNS 고정 + IP 차단)
5
외부 콘텐츠
Fetched URLs / Emails / Webhooks
  • 외부 콘텐츠 래핑 (XML 태그)
  • 보안 알림 주입
